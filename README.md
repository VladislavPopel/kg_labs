Отчет по лабе 2   (в доработке):

Приложение показывает реализацию морфологической обработки и Реализацию высокочастотных фильтров (увеличение резкости).

1) Математическая морфология подход к обработке изображений, суть которого заключается в том, что исходное изображение рассматривается как множество, и к нему применяются теоретикомножественные операции.
Наиважнейшим понятием морфологических преобразований есть понятие структурирующего элемента (структурообразующего множества или примитива), который представляет собой определенную совокупность элементов изображения с одним выделенным (центральным) элементом. 

Морфологические преобразования в лабораторной: • Эрозия • Дилатация • Размыкание • Замыкание • Морфологический градиент
